навички

Маю 4 роки досвіду веб разработки на PHP і Java. З використанням HTML CSS та Javascript. Вмію користуватись при розробці фреймворком YII для PHP, а також бібліотекою jQuery и Bootstrap для розробки клієнтської частини.
Професійно володію комп’ютером та програмами: Word, Excel, Power point, а також вмію користуватись командними оболонками: Ms. DOS, Linux, FreeBSD. Можу самостіійно підняти сервер, або полагодити будь-які програмні неполадки.
